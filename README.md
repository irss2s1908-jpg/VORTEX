 AI Lead Qualification & Outreach App

 Live App:
 https://p95-engine-40d9bd-fm9nv.thinkroot.app/
 
 Description:
This app helps find potential customers using AI, score them based on ICP, and automate mail outreach.

 
An AI-powered pipeline that identifies high-quality enterprise leads, scores them using explainable criteria, and engages them with personalized outreach to drive responses.

## 📌 Problem Statement

P95.AI has a powerful product but lacks a systematic, data-driven approach to identifying and reaching the right enterprise buyers.

This project solves that by building an intelligent lead qualification and outreach system using enriched data and AI-driven personalization.

## 💡 Our Solution

We designed an end-to-end pipeline that:

- Collects 200+ enterprise leads from multiple sources
- Enriches them using and external signals
- Scores leads based on a rigorous ICP framework
- Generates personalized outreach messages using AI
- Implements A/B testing to optimize response rates

## 🔄 System Architecture

Data Sources → Clay → Enrichment → Scoring → Outreach → A/B Testing → Insights

## 🎯 Ideal Customer Profile (ICP)

- SaaS / Tech companies
- 50–1000 employees
- Active engineering teams
- Using cloud infrastructure
- Hiring developers or recently funded

This ensures we target companies with high likelihood of needing AI-driven optimization solutions.

## 📊 Lead Scoring Model

We implemented an explainable scoring system based on:

- Company Fit (30%)
- Tech Stack (25%)
- Growth Signals (30%)
- Decision Maker Presence (15%)

### Classification:
- 🔥 Hot (80–100)
- 🌤 Warm (50–79)
- ❄ Cold (<50)

## 🧠 Explainable Insights

Each lead includes reasoning such as:

- “Hiring 12 engineers → growth signal”
- “Uses AWS → modern infrastructure”
- “Recently funded → expansion phase”

This ensures transparency in scoring decisions.

#Full Business Flow
Lead → Outreach → Reply → Meeting → Deal

#Full Conversion Flow
Raw Companies → Enriched Leads → Qualified Leads → Outreach → Replies → Business Opportunities

## ✉️ Personalized Outreach

We generated tailored cold emails and LinkedIn DMs for top 50 leads.

Each message includes:
- Company-specific insights
- Relevant pain points
- Clear call-to-action

## 🧪 A/B Testing Strategy

For top 20 leads, we created two variants:

- Version A: Pain-point driven
- Version B: Opportunity-driven

### Example Hypothesis:
Pain-based messaging may perform better for fast-scaling startups due to urgency.
